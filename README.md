# Selenium Automation Project

## Project Overview

This project demonstrates web automation testing using Selenium WebDriver, Java, Maven, and TestNG.

The automation scenarios are implemented on the SauceDemo website.

## Tools Used

* Selenium WebDriver
* Java
* Maven
* TestNG
* Git
* GitHub

## Test Scenarios

### Login Test

* Verify successful login with valid credentials.

### Add To Cart Test

* Verify product can be added to shopping cart.

### Checkout Test

* Verify complete checkout process.

### Logout Test

* Verify user can logout successfully.
